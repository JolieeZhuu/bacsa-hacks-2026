
# The Tail of Two Sequences and Fingerprints
A forensic dashboard web application that cross-references DNA, fingerprints, and biometric data to generate ranked suspect lists and a case report using GeminI

## Running the code
1. Clone the repository. If you want to test our program with Gemini API, please create a `.env` file in `/api`, and declare variable `GEMINI_API_KEY=...`.
2. Make sure you are in the same directory as `docker-compose.yml`. Run `docker-compose up --build`!
3. Now you can access the application on `http://localhost` locally :)
